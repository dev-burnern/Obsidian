## 파일 시스템 관련 명령어
---
### 권한 및 그룹 설정
---
- 리눅스 시스템의 모듣 파일과 디렉터리에는 접근권한과 소유권이 부여됨.
- 접근권환은 permission이며 소유권은 ownership임.

명령어 **`ls -l`** 은 파일 속성을 나타냄.
**`- rw-rw-r-- 1 burnern burnern 53 2025-05-19 09:00 TST`**
위와 같은 결과물을 도출함.

| 속성 값        | 의미                                        |
| ----------- | ----------------------------------------- |
| - rw-rw-r-- | - 는 파일 허가권(permission)<br>뒤의 줄은 파일 접근 권한임 |
|             |                                           |
