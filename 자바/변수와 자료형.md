## 🧩변수 (Variables)
### 📌 기본 개념
- **물 컵 비유** 🥤
    - 변수 = 컵 (데이터 저장소)
    - 값 = 컵에 담긴 물
    - `int cup = 300;` ➡️ 300ml 물이 담긴 컵
### 📝 선언 & 초기화
```java
// 🎯 선언만 하는 경우 
int speed; 
// 🎯 선언과 동시 초기화 
int time = 20; 
// 🎯 다중 변수 선언 
int x = 5, y = 10;
```
### ⚠️ 변수명 규칙
1. 영문자/숫자/`_`/`$` 조합 ✅
2. 숫자 시작 ❌ (`3days` 불가
3. 공백 포함 ❌ (`my Var` 불가)
    
4. 예약어 사용 ❌ (`int class;` 불가)
