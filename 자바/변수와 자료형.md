## 🧩변수 (Variables)
### 📌 기본 개념
- **물 컵 비유** 🥤
    - 변수 = 컵 (데이터 저장소)
    - 값 = 컵에 담긴 물
    - `int cup = 300;` ➡️ 300ml 물이 담긴 컵
### 📝 선언 & 초기화
```java
// 🎯 선언만 하는 경우 
int speed; 
// 🎯 선언과 동시 초기화 
int time = 20; 
// 🎯 다중 변수 선언 
int x = 5, y = 10;
```
### ⚠️ 변수명 규칙
1. 영문자/숫자/`_`/`$` 조합 ✅
2. 숫자 시작 ❌ (`3days` 불가)
3. 공백 포함 ❌ (`my Var` 불가)
4. 예약어 사용 ❌ (`int class;` 불가)
## 🔡 자료형 (Data Types)
### 📊 기본 자료형 비교

| 유형  | 키워드       | 크기     | 예시 값       | 범위                   |
| --- | --------- | ------ | ---------- | -------------------- |
| 정수형 | `int`     | 4 Byte | `42`       | -2^31 ~ 2^31-1       |
| 실수형 | `double`  | 8 Byte | `3.141592` | ±4.9e-324 ~ ±1.8e308 |
| 문자형 | `char`    | 2 Byte | `'A'`      | 유니코드 문자              |
| 논리형 | `boolean` | 1 bit  | `true`     | true/false           |
