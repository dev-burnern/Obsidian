## 목차
## 🔢 연산자 (Operators)
## 📌 산술 연산자
```java
int a = 10 + 5;  // ➕ 덧셈 
int b = 20 - 3;  // ➖ 뺄셈 
double c = 7 * 2.5;  // ✖️ 곱셈 
double d = 15 / 4.0;  // ➗ 나눗셈 
int e = 15 % 4;  // 🧮 나머지
```
**🚨 주의**: 정수/정수 연산 시 소수점 버림 (`15/4=3`)

## 📈 관계 연산자

| 연산자  | 의미  | 예시       | 결과      |
| ---- | --- | -------- | ------- |
| `==` | 같음  | `5 == 5` | `true`  |
| `!=` | 다름  | `5 != 3` | `true`  |
| `>`  | 큼   | `10 > 5` | `true`  |
| `<`  | 작음  | `3 < 2`  | `false` |
