## 목차
## 🔢 연산자 (Operators)
## 📌 산술 연산자
```java
int a = 10 + 5;  // ➕ 덧셈 
int b = 20 - 3;  // ➖ 뺄셈 
double c = 7 * 2.5;  // ✖️ 곱셈 
double d = 15 / 4.0;  // ➗ 나눗셈 
int e = 15 % 4;  // 🧮 나머지
```
**🚨 주의**: 정수/정수 연산 시 소수점 버림 (`15/4=3`)

##  관계 연산자
| 유형  | 키워드       | 크기     | 예시 값       |     |
| --- | --------- | ------ | ---------- | --- |
| 정수형 | `int`     | 4 Byte | `42`       |     |
| 실수형 | `double`  | 8 Byte | `3.141592` |     |
| 문자형 | `char`    | 2 Byte | `'A'`      |     |
| 논리형 | `boolean` | 1 bit  | `true`     |     |
