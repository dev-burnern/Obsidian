**값 없음 표시**:

- Python에서는 `None`이 값이 없거나 초기화되지 않은 상태를 나타냅니다.

```
def no_return():
    pass

print(no_return())  # 출력: None
```

| **특징**    | **Python (`None`)** | **다른 언어 (`Null`)**    |
| --------- | ------------------- | --------------------- |
| **타입**    | 객체 (`NoneType`)     | 참조형 변수의 비어 있음 표시      |
| **비교 방식** | is None             | == null               |
| **사용 환경** | Python              | C, Java, SQL 등 다양한 언어 |
