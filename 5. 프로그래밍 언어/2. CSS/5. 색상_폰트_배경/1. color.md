# color

글자 색상을 지정하는 속성입니다.

## 예시
```css
p {
  color: #3498db;
}
```

## 주요 색상 지정 방식
- **이름**: red, blue, green 등
- **HEX**: #ff0000, #00ff00 등
- **RGB**: rgb(255,0,0)
- **RGBA**: rgba(255,0,0,0.5) (투명도)
- **HSL**: hsl(120, 100%, 50%)

---

## 실무 팁
- HEX, RGB, HSL 등 다양한 방식 중 팀/프로젝트 기준을 정해 일관성 있게 사용하세요.
- 투명도(알파값)가 필요한 경우 RGBA, HSLA를 활용하세요.
- 색상 변수(CSS 변수, SCSS 변수 등)로 관리하면 유지보수에 유리합니다.
