# DOM 조작

자바스크립트로 HTML 요소를 추가, 삭제, 변경할 수 있습니다.

## 예시 코드
```javascript
// HTML: <ul id="list"></ul>
const ul = document.getElementById('list');
const li = document.createElement('li');
li.textContent = '새 항목';
ul.appendChild(li);
```

## 팁
- querySelector, querySelectorAll로 CSS 선택자처럼 요소를 찾을 수 있습니다.
