# 비동기 프로그래밍

자바스크립트는 비동기 처리를 위해 콜백, 프로미스, async/await를 지원합니다.

## 예시 코드
```javascript
function delay(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}
async function run() {
  console.log('Start');
  await delay(1000);
  console.log('End');
}
run();
```

## 팁
- 비동기 코드는 async/await로 작성하면 가독성이 좋아집니다.
