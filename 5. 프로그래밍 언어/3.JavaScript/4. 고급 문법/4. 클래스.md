# 클래스

ES6부터 클래스(class) 문법이 도입되어 객체지향 프로그래밍이 쉬워졌습니다.

## 예시 코드
```javascript
class Person {
  constructor(name) {
    this.name = name;
  }
  greet() {
    console.log('Hi, ' + this.name);
  }
}
const p = new Person('Alice');
p.greet(); // Hi, Alice
```

## 팁
- 클래스는 프로토타입 기반을 감싼 문법적 설탕(syntax sugar)입니다.
