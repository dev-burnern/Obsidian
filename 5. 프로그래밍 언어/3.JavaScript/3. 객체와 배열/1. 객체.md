# 객체

객체(Object)는 여러 값을 하나로 묶어서 관리할 수 있는 자료형입니다. 속성(프로퍼티)과 값의 쌍으로 구성됩니다.

## 예시 코드
```javascript
let person = {
  name: 'Tom',
  age: 25,
  greet: function() {
    console.log('Hello!');
  }
};
console.log(person.name); // Tom
person.greet(); // Hello!
```

## 팁
- 객체의 속성에 접근할 때는 `객체.속성명` 또는 `객체['속성명']`을 사용할 수 있습니다.
