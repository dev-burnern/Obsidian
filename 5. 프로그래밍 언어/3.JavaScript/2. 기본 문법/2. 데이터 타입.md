# 데이터 타입

자바스크립트의 주요 데이터 타입:
- 문자열(String)
- 숫자(Number)
- 불리언(Boolean)
- null
- undefined
- 객체(Object)
- 배열(Array)

## 예시 코드
```javascript
let str = 'hello';
let num = 10;
let isOk = true;
let nothing = null;
let notDefined;
let obj = {name: 'Bob'};
let arr = [1, 2, 3];
```

## 팁
- typeof 연산자로 타입 확인 가능
