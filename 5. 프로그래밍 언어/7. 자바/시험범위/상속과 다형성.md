## 상속
### 필요성
- 기존 클래스의 기능(필드, 메서드) 사용하여 새 클래스 만드는 기술
- 다른 클래스의 특징(멤버 메서드와 변수)을 가져옴 **객체지향 프로그래밍 필수**
- 클래스 간 관계 이해, 가독성, 해석 가능성 향상
- 응용 프로그램의 유지·관리에 유용

### 개념
- 클래스 간 관계 설정, 계층적 순서로 정보 관리, 코드 재사용성 상승
- 클래스 생성 시, 원하는 코드 일부가 포함된 클래스가 있다면 -> 상속
