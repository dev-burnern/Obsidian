HTML5에서 텍스트를 표현하거나 꾸미는 주요 태그들을 정리합니다.

| 태그                | 설명                       |
|---------------------|----------------------------|
| &lt;h1&gt; ~ &lt;h6&gt;        | 제목(계층적 구조)           |
| &lt;p&gt;                | 문단                       |
| &lt;b&gt;                | 굵게(단순 시각)             |
| &lt;strong&gt;           | 굵게+의미 강조              |
| &lt;i&gt;                | 기울임(단순 시각)           |
| &lt;em&gt;               | 기울임+의미 강조            |

각 태그별 상세 설명은 하위 md 파일에서 확인하세요.
