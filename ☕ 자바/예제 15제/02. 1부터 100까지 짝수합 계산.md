1부터 100까지 짝수합을 계산하시오.
```java
public class Problem02 {
    public static void main(String[] args) {
        int sum = 0; // 합계를 저장할 변수 초기화
        
        // 1부터 100까지 반복
        for (int i = 1; i <= 100; i++) {
            // 짝수인지 확인 (2로 나누어 떨어지면 짝수)
            if (i % 2 == 0) {
                sum += i; // 짝수면 합계에 더함
            }
        }
        
        // 결과 출력
        System.out.println("1부터 100까지 짝수 합: " + sum);
    }
}
```
