Scanner클래스를 이용하여 두 실수를 입력 받고 입력받은 두수의 사칙연산(+,-,* ,/)을 하고 그 결과를 출력하시오.
```java
import java.util.Scanner; // 키보드 입력을 위한 Scanner 클래스 불러오기

public class Problem01 {
    public static void main(String[] args) {
        // Scanner 객체 생성 (키보드 입력을 받기 위함)
        Scanner scanner = new Scanner(System.in);
        
        // 사용자에게 두 실수 입력 요청
        System.out.print("두 수를 입력하시오(예: 10.5 3.14): ");
        
        // 두 실수 입력 받기
        double num1 = scanner.nextDouble(); // 첫 번째 실수 입력
        double num2 = scanner.nextDouble(); // 두 번째 실수 입력
        
        // 사칙연산 수행 및 결과 출력
        System.out.println(num1 + "+" + num2 + "=" + (num1 + num2)); // 덧셈
        System.out.println(num1 + "-" + num2 + "=" + (num1 - num2)); // 뺄셈
        System.out.println(num1 + "*" + num2 + "=" + (num1 * num2)); // 곱셈
        System.out.println(num1 + "/" + num2 + "=" + (num1 / num2)); // 나눗셈
        
        // Scanner 자원 해제
        scanner.close();
    }
}

```
